Git

Git'as yra populiariausia kodod versijavimo programa.

Git komandos

  • init - sukuria git'o repozitorija(naud viena karta)
  • add - prideda nurodytus failus prie commit'ui paruostu failu(staged)
  • status - nurodo informacija apie tavo lokalia git'o repozitorija(jos statusa)
  • commit -m "zinute" - sukuria kodo versija is stage'intu failu ir pridedazinute apie pakitimus (zinute bandyti islaikyti ne ilgesne nei 50 simboliu)
  • push - is lokalios repozitorijos commit'u nusiuntimas i isorine repozitorija
  • pull - is isorines repozitorijos commit'u atsisiuntimas i lokalia repozitorija
  • remote add origin https://... - isorines repozitorijos prijungimas prie lokalios repozitorijos
  • log - commit'u sarasas
  • chechout branchName|commitID - pereiti i kita kodo versija|saka
  • q - isejimas is logo
  • branch naujosSakosPavadinimas - sukurti nauja saka
  • git checkout + Tab kelis kartus|Enter- isveda kelias sakas(dar git branch -l - isvardina)
